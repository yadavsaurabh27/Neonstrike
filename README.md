# Neon Breach FPS

A browser-based first-person arena shooter prototype built with Three.js.

## Run

The local server is already running at:

```text
http://127.0.0.1:5173/
```

If you need to restart it later:

```bash
python3 -m http.server 5173
```

## Controls

- `WASD`: move
- `Mouse`: aim
- `Click`: fire
- `R`: reload
- `Shift`: sprint
- `Esc`: release mouse

## Notes

The game uses Three.js from a CDN, so the browser needs internet access the first
time it loads the 3D engine.
